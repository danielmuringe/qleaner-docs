# Prerequisites

Before you start contributing to Qleaner, ensure you have the following installed on your machine:

## Node.js

We recommend using **Node.js version 20.0 or higher**. You can check your version by running:

```bash
node -v
```

If you need to install or upgrade Node.js, visit the [official Node.js website](https://nodejs.org/).

## Yarn

We use **Yarn** for package management. You can install it globally via npm:

```bash
npm install -g yarn
```

Check your Yarn version:

```bash
yarn -v
```

## Git

You will need **Git** to clone the repository and manage version control.

```bash
git --version
```

## Repository

Ensure you have access to the [Qleaner repository](https://github.com/qhalahq/qleaner).
